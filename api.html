API

Пользователи

Конечная точка для получения списка всех пользователей.

URL-адрес: /api/users/

Метод Запрос Ответ
GET HTTP_200_OK
  • last_login, nickname, role, first_name, registration_status, user_online_date, last_name, email

Создание пользователя

URL-адрес: /auth/users/

Метод Запрос Ответ
POST
  • 'Content-Type': 'multipart/form-data'
  • username, email,password
 HTTP_201_CREATED

Username
150 characters or fewer. Letters, digits and @/./+/-/_ only.

Password
Your password can’t be too similar to your other personal information. Your password must contain at least 8 characters. Your password can’t be a commonly used password. Your password can’t be entirely numeric.

Вход пользователя

URL-адрес: /auth/token/login/

Метод Запрос Ответ
POST
  • 'Content-Type': 'multipart/form-data'
  • username, password
  • HTTP_200_OK, auth_token

Проверка пользователя

URL-адрес: /auth/users/me/

Метод Запрос Ответ
GET Authorization 200

Authorization
Token <auth_token>

Изменить пароль

URL-адрес: /auth/users/set_password/

Метод Запрос Ответ
POST
  • Authorization
  • 'Content-Type': 'multipart/form-data'
  • current_password, new_password
 HTTP_204_NO_CONTENT

Данные пользователя

URL-адрес: /api/user/

Метод Запрос Ответ
GET username HTTP_200_OK
  • last_login, nickname, role, first_name, registration_status, user_online_date, last_name, email
GET Authorization HTTP_200_OK
  • last_login, nickname, role, first_name, registration_status, user_online_date, last_name, email
PATCH
  • Authorization
  • 'Content-Type': 'multipart/form-data'
  • first_name, last_name, nickname
 HTTP_201_CREATED, HTTP_204_NO_CONTENT

Метод Запрос Ответ
POST Authorization
  • HTTP_200_OK
  • HTTP_204_NO_CONTENT
